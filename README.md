### `ignore-existing-file.sh`

Shortcut for removing and ignoring a tracked file from a Git repository.

### Usage

| Flag                | Description                                         |
| ------------------- | --------------------------------------------------- |
| `-g`, `--gitignore` | Update `.gitignore` instead of `.git/info/exclude`. |

#### Examples

```bash
ignore-existing-file.sh -p scratch.txt
```
